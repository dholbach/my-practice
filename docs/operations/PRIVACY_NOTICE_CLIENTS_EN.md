# Privacy Notice

*Single-practitioner therapy practice*

**Last updated**: May 2026

---

## Summary

Your data is collected as sparingly as possible, stored exclusively on local hardware,
and not shared with third parties — except where explicitly described below.

---

## What data is processed and where?

### Email

| | |
|---|---|
| **What** | Sender, subject, message content |
| **Where** | Proton Mail (end-to-end encrypted) |
| **Third party** | Proton AG, Geneva — [proton.me/security](https://proton.me/security) |
| **How long** | Until manually deleted |

Emails between you and the practice are end-to-end encrypted if you use Proton Mail.
With other providers, transport (TLS) is encrypted but not storage at rest.

---

### Phone

| | |
|---|---|
| **What** | Your phone number; call notes if relevant |
| **Where** | Stored locally in the practice system |
| **Third party** | None |
| **How long** | Duration of therapy + 10 years (§ 147 AO) |

---

### Calendar entries

| | |
|---|---|
| **What** | Appointment code (e.g. "AG", "KC"), date, time — no name |
| **Where** | Google Calendar |
| **Third party** | Google LLC — [Google Workspace Privacy](https://workspace.google.com/terms/dpa_terms.html) |
| **How long** | Until the entry is deleted |

Calendar entries contain only a short code, date and time.
Your name does not appear in the calendar.

---

### Bank transfer

| | |
|---|---|
| **What** | Your name and amount in the bank statement |
| **Where** | Bank statement processed locally — no third party |
| **Third party** | None (the bank itself is outside this scope) |
| **How long** | 10 years (§ 147 AO tax law) |

Matching a bank transfer to your client record happens exclusively on the local
practice laptop.

---

### Clinical records and session notes

| | |
|---|---|
| **What** | Session data, progress notes, diagnostic notes |
| **Where** | Local, encrypted (full-disk encryption + field-level encryption) |
| **Third party** | None |
| **How long** | Duration of therapy + 10 years |

The practice software runs exclusively on the local practice laptop — no cloud server,
no internet access for the application. Clinical notes are encrypted at field level in
addition to full-disk encryption.

---

### Invoices and financial data

| | |
|---|---|
| **What** | Name, address, services rendered, amounts |
| **Where** | Stored locally; invoice delivered by email |
| **Third party** | Only your email provider on delivery |
| **How long** | 10 years (§ 147 AO) |

---

### it's complicated (appointment booking)

| | |
|---|---|
| **What** | Name, email, booked appointment |
| **Where** | it's complicated platform |
| **Third party** | it's complicated — their own privacy policy applies |
| **How long** | According to their policies |

If you book an appointment via it's complicated, their terms and privacy policy apply.
The practice receives only the appointment confirmation.

---

### SMS — appointment reminder / operational notice (planned, opt-in only)

| | |
|---|---|
| **What** | Your mobile number + short generic text (no clinical content) |
| **Where** | seven.io, Berlin |
| **Third party** | seven.io GmbH — GDPR-compliant, ISO 27001 |
| **How long** | Until consent is withdrawn |

This feature is not yet active. When introduced, you will receive a separate notice
and can actively opt in. No SMS will be sent without your explicit consent.

seven.io receives only your mobile number and the message text. No name, no clinical
content, no further contact details are transmitted.

---

## Principles: privacy by design

- No name in calendar — codes only
- No cloud application for clinical data
- Clinical notes additionally encrypted
- New communication channels only with explicit consent
- No disclosure to third parties without legal obligation

---

## Your rights (GDPR Art. 15–21)

- **Access** — What data is stored about me?
- **Rectification** — Correct inaccurate data
- **Erasure** — Where no legal retention obligation applies
- **Restriction** — Restrict processing
- **Objection** — Object to certain processing
- **Portability** — Receive data in machine-readable format

Please direct any requests directly to the practice.
